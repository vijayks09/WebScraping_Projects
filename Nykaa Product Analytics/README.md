Nykaa Product Analytics (Web Scraping + Power BI)

This project focuses on analyzing product data from Nykaa’s Makeup and Skincare categories to understand pricing strategies, discount patterns, and customer behavior.
Data was collected by scraping 1000+ product records using Python (Selenium, BeautifulSoup) and structured into a clean dataset with key features such as price, MRP,
discount %, ratings, and reviews.

The dataset was processed using Pandas and NumPy for data cleaning, transformation, and feature engineering. Exploratory Data Analysis (EDA) was performed to identify
patterns and relationships between pricing, discounts, and customer engagement. Power BI dashboards were developed to visualize insights, including pricing
distribution, brand performance, customer ratings, and product popularity.

The analysis revealed that mid-range pricing dominates the market, discounts increase visibility but not always ratings, and products with higher reviews tend to
build stronger customer trust. This project demonstrates end-to-end data analytics skills including web scraping, data preprocessing, visualization, and business
insight generation, making it highly relevant for Data Analyst and Business Analyst roles.
